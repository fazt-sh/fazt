The Philosophy

Simplicity is the ultimate sophistication.

One Binary

No apt-get, no Docker images to pull. Just a single executable `fazt` that contains everything: web server, database engine, runtime, and assets.

One Database

The filesystem is dead. User sites, configurations, logs, and SSL certificates all live in `data.db`. Backup this one file, and you've backed up your entire world.

Zero External Deps

We believe in self-containment. Fazt doesn't need systemd, nginx, or certbot. It manages its own processes, certificates, and traffic.
